# 校园生活服务小程序

## How to Run

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具，选择"导入项目"
3. 选择项目根目录下的 `frontend-mp` 文件夹
4. AppID 可使用测试号或填入自己的 AppID
5. **重要：在开发者工具右上角"详情" -> "本地设置"中，勾选"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"**
6. 点击"编译"即可预览运行

> ⚠️ 注意：项目中部分图片和音频使用了外部网络链接，原因如下：
> - 微信小程序包大小限制为 2MB，无法内置大量图片和音频资源
> - `wx.previewImage` 图片预览功能仅支持网络图片，不支持本地图片
> - 校园广播使用 `InnerAudioContext` 实现音频播放，网络音频便于演示和测试
> - 图片使用 picsum.photos 随机图片服务，音频使用网易云音乐外链
> - 本项目为演示项目，使用免费公开资源便于快速开发和测试
> - **真实开发时，请将资源链接替换为自己服务器的资源地址**

## Services

本项目为纯前端小程序，数据存储在本地 Storage 中，无需后端服务。

主要功能模块：
- 失物招领：发布/浏览寻物启事和失物招领信息
- 二手市场：发布/浏览二手商品交易信息
- 校园风光：展示校园风景图片
- 校园广播：校园资讯和公告
- 个人中心：用户登录/注册、收藏、浏览历史

## 题目内容

（一）基础功能（必做） 
1.首页模块（对应界面设计、组件使用知识点） 
①　顶部导航栏：显示小程序名称“校园生活服务”，配置导航栏背景色、标题样式（参考第2章app.json的window配置）。 
②　校园公告轮播：使用swiper组件实现公告滚动展示，支持点击查看详情（参考第3章swiper组件）。 
③　分类导航：采用flex布局实现3-4个功能分类（如 “失物招领”“二手市场”“校园风光”“校园广播”），点击跳转对应模块（参考第3章flex布局、第4章navigator组件）。 
④　校园动态：使用scroll-view组件展示最新校园动态（文字+图片），支持下拉刷新（参考第3章scroll-view、下拉刷新配置）。 
2.校园服务模块（对应组件、数据存储知识点） 
(1)  失物招领功能： 
①　发布功能：使用form表单、input、picker组件（选择物品类型、丢失地点）、button组件提交数据，数据存储到本地缓存（参考第4章form组件、第5章本地缓存API）。 
②　列表展示：使用wx:for列表渲染展示失物信息，支持点击查看详情（参考第2章列表渲染、第4章scroll-view）。 
(2)  二手市场功能： 
①　商品发布：支持上传商品图片（wx.chooseImage）、输入商品信息，数据存储到文件系统（参考第5章文件操作API）。 
②　商品筛选：使用picker组件实现价格区间、商品分类筛选（参考第4章picker组件）。 
3.多媒体模块（对应第6章多媒体知识点） 
①　校园风光展示：使用swiper组件实现图片轮播，点击图片支持全屏预览（wx.previewImage）（参考第3章swiper、第5章图片API）。 
②　校园广播：集成音频播放功能，支持播放、暂停、进度显示（使用InnerAudioContext）（参考第6章普通音频API）。 
4.个人中心模块（对应组件、数据存储知识点） 
①　个人信息展示：显示用户昵称、头像（wx.getUserInfo），支持编辑个人资料（picker选择出生日期、地区选择器）（参考第4章picker、省市区选择器）。 
②　收藏功能：支持收藏失物招领、二手商品信息，使用本地缓存存储收藏列表（参考第5章缓存API）。 
③　历史记录：展示浏览过的服务信息，支持清空记录（wx.removeStorage）（参考第5章缓存删除API）。 
（二）进阶功能（选做） 
①　云开发集成：使用云数据库替代本地缓存，实现数据云端存储、实时同步（参考第8章云开发知识点），支持多人协作发布信息。 
②　定位功能：集成位置API（wx.getLocation、wx.chooseLocation），实现失物招领地点定位、二手交易地点显示（参考第5章位置API）。 
③　分享功能：支持将服务信息、校园风光分享给好友或群聊（使用onShareAppMessage）（参考第4章路由与分享API）。 
④　留言互动：在商品详情、失物招领页面添加留言功能，支持实时查看留言列表（参考第4章表单组件、云数据库）。 
⑤　夜间模式：实现明暗主题切换，通过数据绑定控制页面样式（参考第2章数据绑定、第3章样式动态修改）。根据上述要求制作一个微信小程序。

## 项目介绍

校园生活服务小程序是一款为校园师生提供便捷服务的微信小程序，主要功能包括：

### 功能特性

- **失物招领**：发布和查找丢失物品信息，帮助失主找回物品
- **二手市场**：校园内二手物品交易平台，支持发布和浏览商品
- **校园风光**：展示校园美景，记录校园生活
- **校园广播**：发布校园公告和资讯
- **个人中心**：用户管理、收藏夹、浏览历史等

### 技术栈

- 微信小程序原生开发
- 本地 Storage 数据存储
- CSS 变量实现主题定制

### 项目结构

```
frontend-mp/
├── assets/          # 静态资源
├── components/      # 公共组件
├── config/          # 配置文件
├── pages/           # 页面文件
├── services/        # 数据服务层
└── utils/           # 工具函数
```
